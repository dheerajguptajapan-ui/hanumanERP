# Hanuman Enterprise Solution — Local Installation Guide

To run your Hardware ERP locally on your Windows machine, follow these steps:

### 1. Launch the Application
You can launch the application by double-clicking the **START_HERE.bat** file located in this folder.
- **Location**: `c:\Users\trajk\OneDrive\Desktop\inventory managment syestem\hardware-erp\START_HERE.bat`

### 2. Desktop Shortcut
I have created a shortcut on your desktop named **"Hanuman Enterprise Solution"**. You can use this to launch the app anytime.

### 3. Requirements
- **Node.js**: Ensure you have Node.js installed. If not, the launcher will provide a download link.
- **Browser**: The app will automatically open in your default browser at `http://localhost:3000`.

### 4. Updating
Whenever you want to update the app with new changes, simply run the launcher. It will automatically check for dependencies and rebuild the system if necessary.

### 5. Standalone Desktop App (Windows)
I have built a standalone Windows application for you. This is a dedicated app window that doesn't require a browser.
- **App Location**: `c:\Users\trajk\OneDrive\Desktop\inventory managment syestem\hardware-erp\release\win-unpacked\Hanuman Enterprise Solution.exe`
- **Shortcut**: Look for **"Hanuman ERP Desktop"** on your desktop.
